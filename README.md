
# TCGA-LIHC Ketone Body Metabolism Analysis

R scripts for the TCGA-LIHC transcriptomic analysis associated with:
"Ketone body metabolism in MASLD-associated hepatocellular carcinoma" 

## Requirements
R 4.5.2, TCGAbiolinks, survival, ggplot2, ggpubr, survminer, maxstat, estimate

## Data
TCGA-LIHC data downloaded via TCGAbiolinks from GDC Data Portal (open access).

## Usage
Run TCGA_LIHC_ketone_analysis.R sequentially. 
Set working directory and output paths as needed.
